# 手机号码正确，无法登录官网或软件提示号码格式不对

英国手机号码长度为11位数字，国际码0044(也叫国际区号+44)，国内区号0占用了1个数字，英国的手机号码唯一标识7占用了1个数字，剩下的9位数字是具体电话号码。
因此，gifgaff的手机号码格式是+44+0+7+9位数字。
在gifgaff官网或App使用手机号码的时候，需要去掉+44，并在手机号前输入国内区号0，所以输入格式为:07xxxxxxxxx，还请特别留意。

另外:

1. 用giffgaff手机号码注册各类APP时候，国际区号选择UK(英国)或+44，去掉国内区号0，输入10位号码7xxxxxxxxx即可

2. 在国内发送短信给英国手机号码，请用0044+7xxxxxxxxx的号码格式，注意44后面去掉0。

**补充知识：**

英国手机号码规则详解
英国手机号码的格式可以分为国内拨打和国际拨打两种形式。英国手机号（无论哪个运营商，如 giffgaff、O2、EE 等）的标准长度都是 11位数字（国内格式）。

以下是具体的格式说明：

# 国内拨打格式（在英国境内）

在英国境内拨打或输入手机号时，必须以 07 开头，总共 11 位数字。

标准格式： 07XXX XXXXXX
示例： 07123 456789

# 国际拨打格式（在英国境外）

从英国以外的国家拨打或在网站注册国际账号时，需要加上英国的国家区号 +44（或 0044），并去掉国内格式中最前面的数字 0。

标准格式： +44 7XXX XXXXXX 或 +447XXXXXXXXX
示例： +44 7123 456789

# 核心特征与注意事项

号段开头： 英国手机号码的有效号段基本上都集中在 071 至 075、以及 077 至 079（例如常见的 074、077 开头）。

注：076 开头通常用于寻呼机（Pagers），070 开头通常是个人随身号（Personal Numbers），这两者一般不作为普通手机号使用。

字数检查：

包含前面的 0 时，总长一定是 11位。
转换为 +44 格式后，不含区号的实际数字是 10位（即 7 后续接 9 位数字）。
