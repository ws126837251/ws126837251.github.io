giffgaff支持2G/3G/4G/5G信号，在中国，它和中国移动、中国联通均有漫游协议，因此可以通过这2家运营商的基站和网络服务进行漫游通信服务。

注：2025年7月份后，giffgaff在全球都不再支持3G网络。

- 由于某些不可抗拒的原因，giffgaff通过中国联通漫游时候，会出现无法呼出电话，也无法发送短信(部分地区可以)，如果有这方面需要的朋友，请根据自己的手机系统，参考以下流程进行运营商网络调整。

# 苹果用户(i0S17.x版本为例)

1. 在桌面找到「设置」图标，点击进入

2. 在首页找到「蜂窝网络」，点击进入

3. 选择 「giffgaff」，点击进入

4. 在页面中找到「网络选择」，如果看到显示「中国联通」，点击进入

5. 点击「自动」按钮右侧的开关，关闭后，稍等2-3分钟，直到出现多个运营商名称的列表

6. 选择「中国移动」

7. 完成


# 安卓用户(以原生谷歌系统为例，国产安卓类似)


1. 在桌面找到「设置」图标，点击进入

2. 在首页找到「双卡/移动网络」，点击进入

3. 在SIM卡列表中选择「giffgaff」，点击进入

4. 在页面中找到「选取运营商」，如果看到显示「中国联通」，点击「自动选择网络」按钮右侧的开关，关闭。

5. 稍等2-3分钟，直到出现多个运营商名称的列表

6. 选择「中国移动4G」，有些非4G手机，可以选「中国移动2G」

7. 完成


# 特殊情况:


部分卡可能会遇到选择中国移动后，信号丢失或不稳定的情况，一般来说，这是因为您手机附近的移动基站存在以下原因导致，

例如:

1. 移动基站信号较弱或者附近没有中国移动的基站覆盖(极其罕见)。

2. 基站负载较高，优先处理本地用户，海外卡用户的信号连接不稳定(例如体育场，旅游景点，交通枢纽等人群密集地区)。

3. 有中国移动的基站，但是频段规划和giffgaff支持的频段不一致，导致没有信号(极少数)。

4. 少数老人机/非智能机所用的信号频段不支持giffgaff的频段。
